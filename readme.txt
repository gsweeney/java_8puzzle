
READ ME
-------
this program finds the fewest number of moves to solve 
an '8 puzzle' with the format:


1 2 3
4 0 5 
6 7 8

-------

1. compile java files: 
	javac Hw2.java State.java

2. run the program: 
	java Hw2

3. type the name of an input file: 
	input1.txt
	input2.txt

4. program outputs fewest number of moves
